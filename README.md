# DearYou
DearYou is a warm, responsive literature platform for readers, writers, and poetry lovers.

## Features
- Responsive UI using CSS Grid & Flexbox
- Dark / Reading Mode
- Persistent Wishlist using localStorage
- Mobile-friendly navigation
- Curated poetry and reading resources

## Tech Stack
- HTML
- CSS
- JavaScript
